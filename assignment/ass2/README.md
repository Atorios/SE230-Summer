#### 易用性
用户培训时间：  
普通教师高效地进行课程的增删改查所需的培训时间约为10分钟。   
普通教师和教务管理员高效地查看统计信息所需的培训时间约为3分钟。   
普通教师和教务管理员高效地查看实时监控画面所需的培训时间约为5分钟。   
教务管理员高效的查看教师信息所需的培训时间约为3分钟。   
图形标准：   
课堂威视的图形界面使用标准应符合IBM的CUA标准。  
#### 可用性需求：  
课堂威视的可用性需求为对用户友好，容易使用，不需要长期培训。

执行者：普通教师或者教务管理员
